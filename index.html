Object Syntax

                object ကို declare လုပ်တဲ့အခါ { } ကိုသုံးပြီး declare လုပ်ရပါတယ်
                eg. const object = { } 
                ဒီ syntax ကို object literal syntax လို့ခေါ်
                Second Syntax 
                new Object ဆိုတဲ့ keyword ကိုသုံးရေး(case sensitive ဖြစ်)
                eg. const object = new Object()
                Third Syntax
                Object.create ဆိုတဲ့ keyword ကိုသုံးရေး(case sensitive ဖြစ်)
                eg. const object = Object.create()

The Use of Object

                object ကို property, function or method တွေကိုတစ်စုတစ်စည်းတည်းထားတဲ့နေရာမှာသုံးပါတယ်

Object

                Object ဆိုတာ variable တစ်ခုပဲ but object က value အများကြီး assign လုပ်နိုင်။
                object အတွင်းက property syntax
                - name:value,name:value
                -value ရဲ့ data type အပေါ်မူတည်ပြီး ရေးပုံကွာသွား
                -char ဆိုရင် '' or " " နှင့်ရေး အဲလို
                အပြည့်အစုံကို အောက်ပါတွင်ဖတ်

Methods in object

                Methods များက objects များလုပ်နိုင်တဲ့ actions တွေပါပဲ။
                ဥပမာ ကားတစ်စီးသည် စက်နှိုးလို့ရသည်၊ မောင်းလို့ရသည်၊ အဲလိုသဘောပါပဲ။
                Methods များကိုတော့ function အနေနှင့် properties တွေထဲမှာ store လုပ်ပါတယ်
